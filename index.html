Was ist CSS?

CSS bedeutet "Cascading Style Sheets". Damit bestimmst du das Aussehen deiner Website, zum Beispiel Farben, Schriftarten und Abstände.

Wie funktioniert CSS?

HTML ist für den Inhalt zuständig. CSS sorgt für das Design.

Selektoren

  • h1 – Überschriften
  • p – Text
  • .klasse – mehrere Elemente
  • #id – ein einzelnes Element

Eigenschaften

  • color – Textfarbe
  • background-color – Hintergrund
  • font-size – Schriftgröße
  • margin / padding – Abstände

Beispiel-Code

HTML:

<h1>Hallo Welt</h1>
<p>Meine Website</p>

CSS:

h1 {
    color: #092043;
}

CSS Code

Hintergrund ändern

body {
    background-color: lightblue;
}

Text größer machen

p {
    font-size: 24px;
}

Rahmen hinzufügen

.card {
    border: 3px solid #092043;
}

Schatten

.card {
    box-shadow: 5px 5px 15px rgba(0,0,0,0.2);
}
