Was ist CSS?

CSS bedeutet "Cascading Style Sheets". Damit bestimmst du das Aussehen deiner Website, zum Beispiel Farben, Schriftarten und Abstände.

Wie funktioniert CSS?

HTML ist für den Inhalt zuständig. CSS sorgt für das Design.

Selektoren

  • p – Element-Selektor
  • .klasse – klassen-Selektor
  • #id – ID-Selektor

Eigenschaften

  • color: red; – Textfarbe
  • background-color: yellow; – Hintergrund
  • font-size: 20px; – Schriftgröße
  • margin: 20px; – Abstnd außen
  • padding: 20px; – Abstnd innen
  • border: 2px solid black; - Rahmen

Beispiel-Code

HTML:

<h1>Hallo Welt</h1>
<p>Meine Website</p>

CSS:

h1 {
    color: #092043;
}

CSS Code

CSS und HTML zusammenbinden

< link rel="stylesheet" href="style.css">

Hintergrund ändern

body {
    background-color: lightblue;
}

Text größer machen

p {
    font-size: 24px;
}

Rahmen hinzufügen

.card {
    border: 3px solid #092043;
}

Schatten

.card {
    box-shadow: 5px 5px 15px rgba(0,0,0,0.2);
}

Mehrere Eigenschaften zusammen

p {
  color: blue;
  font-size: 18px;
  background-color: lightgray;
}
